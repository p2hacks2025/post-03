# P2HACKS2025 アピールシート

## プロダクト名
Crestarl

## コンセプト
わたしたちは、今回のテーマ「キラキラ」からスター、アーティストを連想し誰でもスターやアーティストになれる支援アプリを作成しました。
## 対象ユーザ
ターゲット属性（ペルソナ）
- メイン層：10代〜20代の学生や若手社会人
- 趣味： 一人カラオケ、TikTokやYouTubeで「歌ってみた」動画をチェックすること、SNSでの自己表現
- ライフスタイル： 自分の部屋が唯一のステージ。スマホ一つで完結するエンタメを楽しみ、常に「何かで自分を表現したい」という欲求を持っている

ユーザーの悩み
- 自分の声をスマホで録音して聴くと、平坦でショボく聞こえてしまい、憧れのアーティストとの差に絶望している。本格的な「歌ってみた」を投稿したいが、数万円するマイクやオーディオインターフェースなどの機材を買うお金がない。従来の編集ソフトは難しすぎて、ピッチ補正やエコーなどの加工が自分ではできない。

私たちのターゲットは、『才能がない人』ではなく、『機材やお金、スキルのせいで、才能を眠らせている人』です。 ユーザーは自分の歌声の現実と理想の間で悩んでいます。 このプロダクトは、その悩みを解消し、ユーザーが本来持っている輝きを解放します。



## 利用の流れ
プロダクトを起動したり、ランディングした時やその後の流れを書きます。
手順書のようなものではなくてもよいですが、読んだらプロダクトの使い方がわかるものがよいです。
必要に応じて、プロダクトを知るまでの流れも書いてよいでしょう。
- アプリを起動し音声ファイルを選択をタップ、自分のスマホからエフェクトをかけたいmp3、wavファイルを選択する。次かけたいエフェクトのタイプをアコーディオンボタンから選択し実行をタップする。
- エフェクトがかけられた生成後の音声が出力されます。その場で再生することができるので気に入ったら保存をおしてダウンロードしてください！



## 推しポイント
ここにはプロダクトの推しポイントを書きます。
なぜそのプロダクトはほかのプロダクトより優れているのでしょうか？他のプロダクトにはない特徴は何でしょうか？ほかでもないそのプロダクトによってユーザーはどのような嬉しさを得ますか？
- 従来の音声編集ソフトは専門的な知識やスキルが必要で編集が難しい。このプロダクトを使えば、専門的な知識やスキルは必要なく、大雑把にエフェクトを指定するだけで手軽に加工ができる。
